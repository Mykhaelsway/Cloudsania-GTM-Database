## Event Day: Half-Day Agenda

### 9:00am - Registration + Coffee (30 min)

**Setup:**
- Registration table with name tags
- Coffee/tea station
- Networking area
- QR code posters (segmentation quiz)

**QR Scan + 2-Question Quiz:**
- Q1: "What's your role?" (Agency Owner, Developer, Founder)
- Q2: "What's your biggest challenge?" (Multi-site management, Deployment complexity, Cost optimization)
- Redirect to segment-specific landing page post-event

---

### 9:30am - "The Agency Chaos Tax" Presentation (30 min)

**Slides:**
1. **Title:** The Agency Chaos Tax
2. **Problem:** Managing 50 sites = 12 logins, 20 hours/week, $2,000/month DevOps
3. **Pain Points:** Downtime anxiety, client churn, profit margin erosion
4. **Solution:** No-Ops model (Your Cloud + Our Interface)
5. **Case Study:** [Local Agency] cut deployment time by 90%
6. **Transition:** "Let's see it in action..."

---

### 10:00am - Live Demo: Unified Dashboard (30 min)

**Demo Script:**
1. **Login:** Show single dashboard
2. **Multi-Org:** Switch between 3 client organizations
3. **Deployment:** Deploy a WordPress site in 3 clicks
4. **Monitoring:** Show uptime, resource usage across all sites
5. **Billing:** Show client-facing reports

**Key Message:** "Manage 50 sites from one screen. No SSH. No cPanel. No chaos."

---

### 10:30am - HANDS-ON: Connect AWS/DO Account (90 min)

**Facilitators:** Founder + 2 team members circulating

**Steps:**
1. Create Cloudsania account (if not already)
2. Navigate to "Connect Cloud" page
3. Enter AWS IAM credentials or DO API token
4. Verify connection (green checkmark)
5. Deploy first test app

**Success Metric:** 80% of attendees successfully connect their cloud

**Support:** 1-on-1 help for those stuck

---

### 12:00pm - Lunch (60 min)

**Provided:** Jollof rice, chicken, plantain, drinks

**Networking:** Encourage attendees to share their agency challenges

**Founder Availability:** Circulate, answer questions, build relationships

---

### 1:00pm - WORKSHOP: Multi-Org Setup (90 min)

**Hands-On Activity:**
1. Create 2 client organizations
2. Deploy 1 app per organization
3. Set up custom domains
4. Configure environment variables
5. Test deployment rollback

**Framework Applied:** Critical Intervention (Agencies need dashboard proof at consideration stage)

---

### 2:30pm - Advanced: Zero-Downtime Deployments (45 min)

**Topics:**
- Blue-green deployments
- Health checks
- Rollback strategies
- Auto-scaling basics

**Demo:** Live zero-downtime deployment

---
